What is python

Python is a versatile and user friendly coding language.

Using python you can perform all sorts of tasks such as numerical calculations and conditional operations.

a = 2*2
print("yes") if a==4 else print("false")
Why use python

Python is easier to learn compared to some other coding languages.

Defining variables, writing expressions, and comprehending other peoples' code is much easier in python.

coding = "easy" if langauge=="python" else "hard"
